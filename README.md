# Starter-Vue-Project
Starter Project to build portfolio and learn VueJS

# Dependencies
1. [Vue.js](https://vuejs.org/)
2. Vue Router
3. [TailwindCSS](https://tailwindcss.com/)
4. [Node.js](https://nodejs.org/en/)

# To Run
1. Go to the project directory
```
> cd vue-project/
```
2. Run development server
```
> npm run dev
```
